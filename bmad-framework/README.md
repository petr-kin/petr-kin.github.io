# BMAD-METHOD Framework Skeleton

## Overview
Systematic AI-driven development framework for my-apps ecosystem based on BMAD-METHOD™ principles.

## Two-Phase Approach

### Phase 1: Agentic Planning
- **Analyst Agent**: Requirements analysis and domain expertise
- **PM Agent**: Project management and scope definition  
- **Architect Agent**: Technical architecture and design decisions

### Phase 2: Context-Engineered Development
- **Scrum Master Agent**: Detailed development story creation
- **Context Engineering**: Full implementation guidance with complete context

## Directory Structure

```
bmad-framework/
├── agents/                 # Agent definitions and prompts
│   ├── analyst/           # Requirements analysis agent
│   ├── pm/               # Project management agent
│   ├── architect/        # Technical architecture agent
│   └── scrum-master/     # Development story agent
├── templates/            # Reusable templates
│   ├── planning/         # Planning phase templates
│   ├── stories/          # Development story templates
│   └── documentation/    # Documentation templates
├── projects/             # Active project workspaces
├── tools/               # Framework utilities
└── config/              # Configuration files
```

## Usage
1. Initialize new project with planning phase
2. Run agentic planning collaboration
3. Generate context-engineered development stories
4. Execute systematic development

## Integration Points
- My-Apps ecosystem (HIVE, Spider, TIS, Analyzer-Kit)
- Claude AI API integration
- Existing project architectures
